## Overview

Customer support analytics platform developed to integrate operational data from APIs and databases into a centralized Business Intelligence environment.

The project combines customer support data extracted from REST APIs and internal SQL databases, processing and transforming raw information into structured datasets suitable for KPI monitoring and operational analysis.

The final output is integrated into Power BI to create interactive dashboards focused on support team performance, operational metrics and business insights.

## Features

- Customer support data extraction from API sources
- Data extraction from internal SQL databases
- Data cleaning and transformation using Python
- Integration of multiple data sources
- KPI calculation and standardization
- Preparation of structured datasets for Power BI
- Support performance dashboard development
- Operational performance monitoring

## Technologies

- Python
- SQL
- Power BI

- ETL
- Data Transformation
- Business Intelligence
- KPI Reporting

## Workflow

- Customer support data is extracted from API sources and internal databases.
- Raw operational data is processed and transformed using Python.
- Data from different sources is combined into structured analytical datasets.
- The processed data is loaded into Power BI.
- Interactive dashboards are created to monitor support performance indicators.

## Dashboard

The Power BI dashboard provides insights into:

- Support volume evolution
- Response time analysis
- Resolution time tracking
- Team performance metrics
- Agent productivity indicators
- Operational KPIs

## Project Purpose

The objective of this project was to develop an automated analytics workflow capable of integrating customer support data from different sources into a centralized Business Intelligence solution.

The project aims to reduce manual reporting tasks, improve KPI visibility and enable managers to make data-driven decisions based on operational performance.
